# JetLink MCP

**Give your AI Assistant the Eyes of IntelliJ.**

JetLink MCP connects your AI coding assistant (Cursor, Claude Desktop, etc.) to IntelliJ's powerful PSI (Program Structure Interface) engine for true semantic code understanding.

## Key Features

- **Semantic Intelligence**: Finds code by meaning, not just text matching
- **Smart Symbol Search**: Instantly locate classes, methods, and fields
- **Call Hierarchy**: Trace callers and callees through your entire codebase
- **Type Hierarchy**: Visualize class inheritance and interface implementations
- **Safe Refactoring**: Rename variables and methods with IDE confidence
- **Token Efficient**: Extracts only relevant code blocks, saving up to 99% tokens
